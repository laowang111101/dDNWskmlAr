# 人事管理系统 java170

## 项目概述

本项目是一个基于Springboot+Vue的前后端分离人事管理系统，采用Javaee技术栈进行开发。系统遵循MVC三层架构设计，结合Spring、SpringBoot、Mybatis框架，以及Vue.js和Maven工具，实现了一套B/S架构的Web应用。

## 技术栈

- 前端：Vue.js
- 后端：Springboot、Javaee、Spring、Mybatis
- 数据库：MySQL
- 项目管理：Maven

## 功能模块

- 用户管理
- 员工信息管理
- 薪资管理
- 考勤管理
- 权限管理

## 系统角色

- 系统管理员
- 人事管理员
- 员工

## 运行环境

- 操作系统：Windows/Linux/MacOS
- 数据库：MySQL（建议版本5.7+）
- JDK版本：1.8+
- Node.js版本：建议使用最新稳定版本

## 部署步骤

1. 搭建开发环境，包括Java、MySQL、Node.js等。
2. 克隆项目代码到本地。
3. 导入MySQL数据库。
4. 前后端分别编译打包。
5. 部署应用到服务器。
